# Library Management System

## Description

The Library Management System is a console-based application developed in **C++** using **Object-Oriented Programming (OOP)** principles. The system provides separate functionalities for **Admin** and **Customer** users. The **Admin** can add, update, remove, and manage library resources, generate library reports, customer reports, and issued/overdue reports. The **Customer** can register an account, log in, search for books and magazines, borrow and return resources, view borrowing history, and manage account balance. The system also uses **File Handling** to store library and customer data and **Exception Handling** to manage invalid operations, making it a complete library management solution.

---

## Features

* Admin Login
* Customer Registration and Login
* Add Library Resources
* Update Library Resources
* Remove Library Resources
* View Available Books and Magazines
* Borrow Resources
* Return Resources
* Search Resources by:

  * Title
  * Category
  * Author
  * Resource ID
* Borrow History Tracking
* Customer Balance Management
* Automatic Fine Calculation for Late Returns
* Generate Library Report
* Generate Customer Report
* Generate Issued and Overdue Report
* File Handling for Data Persistence

## Exception Handling

Custom exception classes are implemented to handle different error conditions, including:

* Insufficient Funds
* Borrow Limit Reached
* Resource Not Available
* Invalid Return Date
* Invalid Resource Operation
* Invalid Report Generation
